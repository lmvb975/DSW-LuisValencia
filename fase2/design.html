Propósito

Este sitio tiene como proposito el fomentar la educación en el ambito web, presentadno una alternativa didactica y efectiva ante la imposibilidad de asistir a las aulas. En particular en este curso trataremos conceptos como lo son la etiqueta HTML, sus atributos y sus valores

Etiquetas Basicas de HTML

Segun ebschool HTML es un markup language, lo que significa que está escrito con códigos que puede leer una persona sin que sea necesario compilarlo primero. En otras palabras, el texto en una página web está «marcado» con estos códigos para dar instrucciones al navegador web sobre cómo mostrar el texto. Estas etiquetas de marcado son las propias etiquetas HTML

Atributos y Valores

Segun desarrolladoresweb Los atributos en HTML se pueden considerar como información adicional que se coloca siempre en la etiqueta de apertura, esto para añadir, complementar información o significado al elemento HTML al que estamos colocando.
